# Setup Files

Los archivos setup son scripts que permiten crear y activar un entorno virtual de virtualenv en los respectivos proyectos.

Para utilizarlos basta con copiar el archivo específico según tu sistema operativo `setup.bat para Windows` y `setup.sh para Linux` en el directorio del proyecto

## Ejecución

Crear entorno virtual desde la CMD en Windows

```cmd
> setup.bat
```

Activar entorno virtual desde la CMD en Windows
```cmd
> setup.bat activate
```

Crear entorno virtual desde la terminal en Linux

```cmd
> setup.sh
```

Activar entorno virtual desde la terminal en Linux
```cmd
> setup.sh activate
```
